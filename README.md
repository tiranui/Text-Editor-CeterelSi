# Text-Editor-CeterelSi

Ini adalah Repository yang berisi text editor buatan sendiri dengan bahasa pemrogaman C

1. Cara Setup bagaimana
2. Instalasi (jika ada)
3. Cara pakai (sertakan contoh atau gambar atau video (tidak wajib))
4. Identitas (NIM, Nama, ID, dan Manager)

# Cara Set Up aplikasi di vs code

1. clone github menggunakan perintah (git clone https://github.com/tiranui/Text-Editor-CeterelSi.git)
2. setelah diclone tekan ctrl+shift ~ (untuk membuka terminal)
3. ketik (gcc main.c Rafli.c Fauzan.c noel.c -o main.exe) pastikan anda sudah menginstal gcc
4. kalau sudah, ketik .\main.exe

# set up aplikasi di cmd

1. setelah kamu clone kamu pergi ke folder dan cari dimana program disimpan
2. setelah itu kamu buka cmd menggunakan Windows+R lalu ketik cmd
3. setelah itu kamu pindah direktori menggunakan cd <direktori dari file program>
4. didalam cmd kamu ketik (gcc main.c Rafli.c Fauzan.c noel.c -o main.exe)
5. jika sudah tinggal ketik main.exe
